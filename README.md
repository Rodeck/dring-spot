# dring-spot 


## Technology overview

1. Backend:
    - .Net (.net core 3.0)
    - Tests:
        - test framework: xUnit
        - mocking library: NSubstitute
2. Frontend:
    - react-native
3. Database:
    - postresql (for now its sql server, needs to be changed to cloud storage)

  -  PostgreSQL
